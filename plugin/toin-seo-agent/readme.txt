Conector REST API para o sistema TOIN SEO Agent. Sem interface de usuário.

== Description ==
Este plugin expõe endpoints REST que permitem ao TOIN SEO Agent ler e escrever
metadados SEO (title, meta description, schema JSON-LD, canonical) nos plugins
Yoast SEO, Rank Math, AIOSEO e SEOPress.

Não tem interface de usuário. Instale e ative — o agente cuida do resto.
